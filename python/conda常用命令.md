* 创建环境

  ```
  conda create --name wtalc python=3.6
  ```

* 激活环境

  ```
  activate wtalc
  ```

* 列出环境

  ```
  conda env list
  ```

  ```
  conda info --env
  ```

* 删除环境

  ```
  conda remove -n wtalc --all
  ```

* 查找包

  ```
  conda search pytorch
  ```

* 安装包

  ```
  conda install python-gflags
  ```

  ```
  conda install pytorch=1.0.1
  ```

* 卸载包

  ```
  conda uninstall pytorch
  ```

* 列出下载的包

  ```
  conda list
  ```
